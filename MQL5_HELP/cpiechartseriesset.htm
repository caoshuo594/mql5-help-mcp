SeriesSet

Sets a series of values that will be shows on the pie chart.

 bool  SeriesSet(
   const double&  value[],  // values
   const string&  text[],   // labels
   const uint&    clr[],    // color
   )

Parameters

value[]

[in] Array of values.

text[]

[in] Array of value labels.

clr[]

[in] Array of value colors.

Return Value
