CharToString

Converting a symbol code into a one-character string.

string  CharToString(
   uchar  char_code      // numeric code of symbol
   );

Parameters

char_code

[in]  Code of ANSI symbol.

Return Value

String with a ANSI symbol.

 

Example:

//| Script program start function                                    |
