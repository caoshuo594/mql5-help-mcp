Mode (Get Method)

Gets the value of "Mode" property (bars, candles, or line).

ENUM_CHART_MODE  Mode() const

Return Value

Value of "Mode" property of the object assigned to the class instance. If there is no chart assigned, it returns WRONG_VALUE.
